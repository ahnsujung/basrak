| 줌 | 대략적 범위 | 비고 |

---

## 모드 전환 기준

| 줌 범위 | 모드 | 설명 |
|---------|------|------|
| 7 ~ 10 | 클러스터 모드 | `leaflet.markercluster` 사용 |
| 11+ | 개별 마커 모드 | 클러스터 해제, 관측점별 렌더 |

- 전환 기준: `disableClusteringAtZoom: 11`

---

## 클러스터 모드 (줌 7~10)

### 클러스터 설정
| 항목 | 값 |
|------|-----|
| maxClusterRadius | 20px |
| disableClusteringAtZoom | 11 |
| spiderfyOnMaxZoom | false |
| zoomToBoundsOnClick | false |
| animate | false |
| 클러스터 아이콘 | 투명 (1×1px) — 도트로 대체 |

### 클러스터 도트 (건수 기준)
| 관측 건수 | 도트 사이즈 |
|----------|-----------|
| 1~3건 | 6px |
| 4~10건 | 12px |
| 11~20건 | 18px |
| 21건+ | 26px |

### 클러스터 펄스
| 항목 | 값 |
|------|-----|
| 크기 | 도트 × 1.5 |
| 색상 | 클러스터 평균 위험도 컬러 |
| 불투명도 | 0.3 |
| 애니메이션 | `cluster-pulse 2s ease-out infinite` |

### 클러스터 영향권 원
| 항목 | 값 |
|------|-----|
| 바깥 원 반경 | 최원점 거리 + 10km |
| 바깥 원 스타일 | weight 0.8, opacity 0.2, fillOpacity 0.06 |
| 안쪽 원 반경 | 바깥 원 × 0.45 |
| 안쪽 원 스타일 | weight 0.5, opacity 0.15, fillOpacity 0.08 |
| 중심 계산 | 관측점 실제 위치 평균 |

### 단독 마커 (클러스터에 속하지 않은 관측)
| 항목 | 값 |
|------|-----|
| 영향권 반경 | 10km (고정) |
| 안쪽 원 | 10km × 0.45 = 4.5km |
| 도트 사이즈 | 8px |
| 펄스 사이즈 | 12px (8 × 1.5) |

---

## 개별 마커 모드 (줌 11+)

### 영향권 반경 (줌별)
| 줌 | 영향권 반경 | 안쪽 원 (×0.45) |
|----|-----------|---------------|
| ≤7 | 5,000m (5km) | 2,250m |
| 8~9 | 3,000m (3km) | 1,350m |
| 10~11 | 1,500m (1.5km) | 675m |
| 12+ | 500m | 225m |

### 영향권 원 스타일
| 원 | weight | opacity | fillOpacity |
|----|--------|---------|-------------|
| 바깥 | 0.8 | 0.25 | 0.06 |
| 안쪽 | 0.5 | 0.18 | 0.08 |

### 마커 사이즈 (줌별)
| 줌 | 마커 전체 크기 | 구성 |
|----|-------------|------|
| ≤7 | 16px | 바깥 원(100%) + 안쪽 원(55%) + 중심 도트(5px) |
| 8~9 | 20px | 동일 구조 |
| 10~11 | 26px | 동일 구조 |
| 12+ | 32px | 동일 구조 |

### 마커 구성 (동심원)
| 요소 | 크기 | 스타일 |
|------|------|--------|
| 바깥 원 | 100% | border 1px solid, opacity 0.5 |
| 안쪽 원 | 55% | border 1.5px solid, opacity 0.75 |
| 중심 도트 | 5px | 위험도 컬러, box-shadow 6px 2px |

---

## 위험도 컬러

| 점수 | 등급 | 색상 |
|------|------|------|
| 1~3 | 낮음 | `#81C784` (녹색) |
| 4~5 | 보통 | `#FFE300` (노란색) |
| 6~7 | 높음 | `#FF6D00` (주황색) |
| 8~10 | 매우 높음 | `#D32F2F` (빨간색) |

---

## 클릭 동작

| 대상 | 동작 |
|------|------|
| 클러스터 도트 | 해당 클러스터 관측들 → SummaryCard 모달 |
| 단독 마커 (클러스터 모드) | SingleCard 모달 |
| 개별 마커 (줌 11+) | 근접 관측 검색 → 1건이면 SingleCard, 2건+ 이면 SummaryCard |
| 근접 판정 기준 | `0.5 / 2^(zoom - 7)` 도 이내 |
