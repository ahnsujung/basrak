## 오늘 한 일

### 1. 지도 안 보임 버그 수정 (`KakaoMap.jsx`)
- **원인**: `h-full` (`height: 100%`)이 0px으로 resolve됨
  - `PageLayout`이 `min-h-dvh`(definite height 아님) → `main`의 `flex-1`도 definite height 아님 → `h-full` = 0
- **수정**: `w-full h-full` → `absolute inset-0` (포지셔닝 기반으로 부모를 채움, definite height 불필요)
- **추가**: `map.invalidateSize()` 호출 (초기 크기 계산 보정)
