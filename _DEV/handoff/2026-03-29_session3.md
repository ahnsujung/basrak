## DB 마이그레이션 (적용 완료)
- `ALTER TABLE observations ADD COLUMN IF NOT EXISTS address text;`
- `CREATE POLICY "users can delete own observations" ON observations FOR DELETE USING (user_id = auth.uid());`
- `risk_score` generated column → 매트릭스 기반 CASE+ARRAY로 변경:
  ```sql
  ALTER TABLE observations DROP COLUMN risk_score;
  ALTER TABLE observations ADD COLUMN risk_score integer
    GENERATED ALWAYS AS (
      CASE
        WHEN dryness_level = 1 THEN (ARRAY[1,1,2,2,3,3])[wind_level]
        WHEN dryness_level = 2 THEN (ARRAY[3,3,4,5,5,6])[wind_level]
        WHEN dryness_level = 3 THEN (ARRAY[5,5,6,7,7,8])[wind_level]
        WHEN dryness_level = 4 THEN (ARRAY[7,7,8,9,9,10])[wind_level]
      END
    ) STORED;
  ```
